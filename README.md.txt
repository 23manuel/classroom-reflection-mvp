# Classroom Reflection MVP

This is a lightweight teacher reflection and classroom engagement tracking system.

## Purpose
To help teachers quickly reflect on daily classes and see simple engagement trends over time, without extra workload.

## Features
- Daily class reflection (5â€“6 inputs)
- Automatic Class Engagement Index (CEI)
- Weekly engagement trends
- Private dashboards per teacher
- Offline-friendly using SQLite

## Tech Stack
- Python
- Streamlit
- SQLite
- Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Start the app:
   streamlit run app.py
